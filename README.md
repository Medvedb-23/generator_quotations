# Генератор цитат

Приложение для управления коллекцией цитат с графическим интерфейсом на PyQt5. Позволяет добавлять, редактировать, удалять цитаты, фильтровать по автору и категории, загружать изображения, переключать тёмную/светлую тему, экспортировать и импортировать данные в CSV.



## Требования

- Python 3.10 или новее.

- Git (для клонирования репозитория).



## Установка и запуск



Следуйте этим шагам, чтобы развернуть проект на своем компьютере:



1. Клонируйте репозиторий:

   `git clone https://github.com/Medvedb-23/generator_quotations.git`

2. Перейдите в папку с проектом:

   `cd generator_quotations`



3. Создайте виртуальное окружение:

   `python -m venv venv`



4. Активируйте виртуальное окружение:

   Для Windows (PowerShell):

&#x20; ```.\\venv\\Scripts\\activate```

&#x20; (Если возникает красная ошибка безопасности, сначала выполните команду `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`, а затем повторите попытку активации).

   Для macOS/Linux:

&#x20; ```source venv/bin/activate```

5. Установите необходимые зависимости:

   `pip install -r requirements.txt`

6. Запустите приложение:

   `python main.py`
